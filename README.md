# README — Sistema de Costeo Estándar

## Descripción
Aplicación web que implementa un sistema de **costeo estándar**, permitiendo calcular y analizar variaciones entre costos reales y presupuestados.  
Basado en los principios contables vistos en clase, el software aplica fórmulas de **precio estándar (PS)** y **cantidad estándar (QS)** para generar reportes de desempeño financiero.

---

## Objetivo
Facilitar la **planeación, control y análisis de costos** mediante un sistema digital con módulos diferenciados y cálculos automatizados.

---

## Stack Tecnológico
**Frontend**
- HTML, CSS, JavaScript (Vanilla)
- TailwindCSS (CDN)
- Chart.js (gráficos)

**Backend**
- Node.js + Express
- `multer` (carga de archivos)
- `xlsx` (procesamiento de hojas Excel)
- Lógica de negocio para costeo y generación de reportes

**Base de datos**
- JSON o CSV local (sin persistencia compleja)

---

## Módulos Principales
1. **Login de usuario**
2. **Carga de archivos Excel**
3. **Procesamiento de costos estándar**
4. **Análisis de variaciones**
5. **Reportes visuales (gráficos y tablas)**

---

## Ejecución local
```bash
# Clonar repositorio
git clone <repo_url>

# Instalar dependencias
npm install

# Iniciar servidor
npm start
````

Abrir en el navegador: **[http://localhost:3000](http://localhost:3000)**
