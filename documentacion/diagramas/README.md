# Diagramas — Documentación de módulos

En este directorio se almacenan los diagramas UML y de flujo correspondientes a cada módulo del sistema de costeo estándar.

## Estructura
- **login/**: Diagramas del módulo de autenticación
- **carga_excel/**: Diagramas del módulo de carga y validación de archivos Excel
- **procesamiento_costos/**: Diagramas del procesamiento de costos estándar
- **analisis_variaciones/**: Diagramas del análisis de variaciones
- **reportes/**: Diagramas del módulo de reportes y visualización

## Tipos de diagramas requeridos por módulo
- Casos de uso
- Clases
- Secuencia
- Estados
- Actividades
- Componentes

Coloca los archivos de cada diagrama (PNG, PDF, PlantUML, etc.) en la carpeta correspondiente.
