# Stack - Costeos Web App

## General
- **Tipo de proyecto:** Web Application  
- **Duración estimada:** 2 semanas  

---

## Frontend
- **Lenguaje:** HTML, CSS, JavaScript (Vanilla)  
- **UI:** TailwindCSS CDN  
- **Gráficos (opcional):** Chart.js  
- **Manejo de datos:** Fetch API para comunicarse con el backend  

---

## Backend
- **Lenguaje:** JavaScript  
- **Framework:** Node.js + Express  
- **Carga de archivos:** `multer`  
- **Procesamiento de Excel:** `xlsx` (SheetJS)  
- **Lógica de negocio:** cálculos de costeo, generación de reportes  

---

## Base de Datos
- JSON local o CSV si no se requiere persistencia compleja  

---

## Despliegue
- **Entorno local:** Node.js + npm scripts
